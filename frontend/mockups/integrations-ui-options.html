Settings → Integrations (UI Option Gallery)

Goal: move GitHub/GitLab out of isolated workspace into a reusable integrations hub.

GitHub GitLab Slack (later) WhatsApp (later)

1) IA / Placement

New top-level in Settings: Integrations. Each provider gets status, auth, scopes, repo binding, and health.

2) Onboarding Flow

Step A auth (OAuth/SSH), Step B verify access, Step C optional repo/project link (skippable), Step D done.

3) Future-Proof Model

Shared provider shell + capability blocks (repos, messaging, webhooks, notifications) for Slack/WhatsApp later.

4) UX Principle

Fast path first. One clear primary button, advanced controls hidden behind “Advanced”. Minimal required fields.
