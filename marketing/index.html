    Overlay mode

    Instant access with a floating overlay.

    The overlay is a feature of the studio. It floats above your apps so you can capture requests, see responses, and keep moving without context switching.

    • Borderless, draggable window that stays on top.
    • Global hotkeys to show or hide the overlay instantly.
    • Auto-reconnects while the local server finishes starting.
    • Built-in widget UI so you can chat immediately.

    Feature matrix

    Everything Intelligravex ships, in one place.

    All features below reflect the current studio and built-in overlay mode.

    Assistant Studio

    • Create, configure, and manage multiple assistants.
    • Per-assistant personality, system prompt, and response style.
    • Central dashboard for assistants, chats, and developer settings.

    Multi-Model Orchestration

    • Main chat model for reasoning and response quality.
    • Voice input and voice output models.
    • Live web search for fresh information.
    • Post-processing model for tool responses.
    • Long-context summary model.
    • Adjustable history window for each assistant.

    Real-Time Conversations

    • Streamed responses (text and audio) with latency metrics.
    • Live mic test and talk loop built into the UI.
    • Continuous session support (start, stop, new conversation).

    Data Agent (Optional)

    • Run each conversation in its own isolated workspace.
    • Long-running tasks persist across turns.
    • Files, logs, and outputs stored per workspace.
    • Multiple concurrent containers for parallel work.
    • Container monitor with status and kill controls.
    • Automatic respawn of the workspace when needed.

    Tools and Integrations

    • Built-in system tools for metadata and web search.
    • Connect external APIs with structured inputs.
    • Validate outputs and keep responses consistent.
    • Map tool results into durable memory.
    • Deterministic reply templates when needed.
    • Optional post-processing to refine tool results.
    • Only mapped fields are saved into memory.

    Host Actions + Screen Capture

    • Run local shell commands or AppleScript on demand.
    • Screen capture with vision summaries.
    • Approval gates available per assistant.

    Context and Memory

    • Conversation metadata can personalize prompts and replies.
    • Automatic summarization as sessions grow.
    • Long conversations stay fast while keeping key facts.

    Local-First and Secure

    • Workspaces, configs, and history stored locally.
    • Provider secrets encrypted at rest.
    • You decide what gets shared externally.

    Embeddable Chat Widget

    • Public text-chat widget you can embed in websites.
    • Client key support and WebSocket transport.

    Git and SSH Tooling

    • Secure repo access for Data Agent workflows.
    • Useful for code tasks, automation, and structured work.

    Overlay Mode (Feature)

    • Borderless overlay window that can float above apps.
    • Global hotkeys for instant access.
    • Loads the built-in widget UI from the bundled server.
    • Auto-reconnects while the server starts.
    • Diagnostic logging for troubleshooting.

    First-Time Setup

    • OpenAI API key required for voice; OpenAI or OpenRouter for chat.
    • Data Agent is optional and needs Docker.
    • Default Data Agent image can be overridden.
    • All core features work without Docker.

    If macOS blocks the app

    If you see “Apple could not verify” after the download, use the steps below.

    1. Open Finder and go to Downloads.
    2. Right-click the Intelligravex app (GravexOverlay) and choose Open.
    3. When the warning appears again, click Open.

    If macOS says the app is damaged, run this once in Terminal:

    xattr -dr com.apple.quarantine ~/Downloads/GravexOverlay.app

    Or: System Settings → Privacy & Security → Open Anyway.
